# Security Policy

## Supported Version

Cabang utama `main` adalah versi yang aktif dipelihara.

## Reporting a Vulnerability

Laporkan celah keamanan secara privat ke maintainer repository sebelum
membuka issue publik. Sertakan:

- Ringkasan celah.
- Langkah reproduksi.
- Dampak yang mungkin terjadi.
- Saran perbaikan jika ada.

Jangan menyertakan data pribadi siswa, dokumen SPMB, credential, atau dump
database dalam laporan.
